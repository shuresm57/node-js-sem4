
# Movies REST API

| Endpoint | Method | Description |
|----------|--------|-------------|
| /movies | GET | Retrieve all movie resources. |
| /movies/{id} | GET | Retrieve a movie resource by id. |
| /movies | POST | Create a movie resource. |
| /movies/{id} | PUT | Update a movie resource. |
| /movies/{id} | PATCH | Update parts of a movie resource |
| /movies/{id} | DELETE | Delete a movie resource. |
